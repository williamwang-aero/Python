# Lecture Notes

Recommended software: VS CODE

To start, type "code hello.py" in Terminal. This tells VS code which file to open, in this case, "hello.py". Then if we type a command, like print("hello, world"), how do we run it? There's no run button in VS CODE, so we will have to type in "python hello.py". The "python" calls the interpreter to run the python command in your computer. The "hello.py" tells the computer which part to run, therefore, it would run and print: hello, world.
